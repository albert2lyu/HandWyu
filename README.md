# HandWyu
本软件是邑大成绩的升级版本。这次界面全新升级，同时功能也发生了很大的变化。功能主要有，查成绩，计算绩点，查看学习情况，导入课程表，校历，查询考试安排，图书查询，公交查询，义工时查询，饭卡丢失平台，同时还有一个简单的英语小词典功能。希望可以给五邑大学的学生带来便捷。同时bug难免会存在的，请大家多多指教。
